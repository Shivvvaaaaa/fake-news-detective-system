# 🕵️ Fake News Detector

## 📌 Description
This project is a web-based application that detects whether a news article is fake or real using machine learning.

## 🚀 Features
- User can enter news text
- System predicts fake or real news
- Uses ML model for detection

## 🛠️ Technologies Used
- HTML, CSS, JavaScript
- Python (Flask)
- scikit-learn

## ▶️ How to Run
1. Run the Flask backend:
   python app.py

2. Open index.html in browser
